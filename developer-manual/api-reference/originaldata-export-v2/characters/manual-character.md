# 埋点事件与变量字段

{% tabs %}
{% tab title="custom_event（埋点事件）" %}
| 名称 | 类型（长度） | 说明 |
| -- | ------ | -- |

| visitUserId | string（64） | <p>&#x3C;访问用户ID></p><p>访问用户的唯一标识，由GrowingIo自动生成。 示例：fc55728b-41ab-42ff-8b1f-714e44c65fd6</p> |
| ----------- | ---------- | -------------------------------------------------------------------------------------------: |

| sessionId | string（50） | <p>&#x3C;访问ID></p><p>当前访问的唯一标识。 示例：6b5099c7-6006-422d-92ac-4f3bf4ddd37c</p> |
| --------- | ---------- | --------------------------------------------------------------------------: |

| time | bigint | <p>&#x3C;时间戳></p><p>事件发生的时间戳。</p><p>示例：1520899220665</p> |
| ---- | ------ | -------------------------------------------------------: |

| sendTime | bigint | <p>&#x3C;发送时间></p><p>请求在SDK发送的时间戳。</p><p>示例：1520899221211</p> |
| -------- | ------ | ------------------------------------------------------------: |

| pageTime | bigint | <p>&#x3C;页面时间></p><p>pvar对应的页面请求的产生时间。</p><p>示例：1516349263375</p> |
| -------- | ------ | ----------------------------------------------------------------: |

| domain | string（100） | <p>&#x3C;域名></p><p>访问的域名，当为 iOS / Android 时，为 app 包名。</p><p>示例：www.growingio.com</p> |
| ------ | ----------- | -----------------------------------------------------------------------------------: |

| page | string（1024） | <p>&#x3C;页面></p><p>用户访问的当前页面。</p><p>示例：pages/index</p> |
| ---- | ------------ | -----------------------------------------------------: |

| queryParameters | string （1024） | <p>&#x3C;查询参数></p><p>当前网站页面URL中的查询参数。</p><p>示例：cid=1234567</p> |
| --------------- | ------------- | -------------------------------------------------------------: |

| eventName | string（50） | <p>&#x3C;事件名称></p><p>自定义事件的标识符。</p><p>示例：recenue</p> |
| --------- | ---------- | ---------------------------------------------------: |

| eventNumber | double | <p>&#x3C;事件数量></p><p>已废弃，请忽略此字段</p> |
| ----------- | ------ | ----------------------------------: |

| eventVariable | map\<string,string> | <p>&#x3C;事件级变量></p><p>自定义事件级变量。</p><p>示例：{"price":"50.0","item":"101"}</p> |
| ------------- | ------------------- | -------------------------------------------------------------------------: |

| loginUserId | string（200） | <p>&#x3C;登录用户ID></p><p>推荐使用那些不能定位到个人的ID信息，通常为企业内部使用的CRM ID。</p><p>示例：user12345</p> |
| ----------- | ----------- | ---------------------------------------------------------------------------------: |

| pageRequestId | string（23） | <p>&#x3C;页面请求ID></p><p>GrowingIO系统内部用于标识一个唯一的页面请求的ID，可以用来关联page 数据。</p><p>示例：1521010820647fa5a9314e6</p> |
| ------------- | ---------- | -------------------------------------------------------------------------------------------------------: |
{% endtab %}
