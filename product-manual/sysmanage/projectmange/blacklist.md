# IP及域名数据过滤

IP 及域名数据过滤可以去除内部人员使用对数据造成的干扰，避免 trackingcode 被拷贝所造成的数据污染。

1. 在界面右上角单击 ![](https://github.com/growingio/growingio-docs-v3/tree/d520f4a494f6c0635c83422f55c665597e79ee96/.gitbook/assets/2019-10-10_18-59-32%20%281%29.png) 选择项目配置，打开项目配置界面。默认打开选择**IP及域名数据过滤**页签。
2. 单击 ![](https://github.com/growingio/growingio-docs-v3/tree/d520f4a494f6c0635c83422f55c665597e79ee96/.gitbook/assets/bian-ji-an-niu.png) 配置**IP黑名单**和需要过滤的**域名**。
3. 配置完成后打开开关。
