# 如何刪除不要的項目？

{% hint style="warning" %}
删除项目是高度危险的操作，为了保证数据安全。  
  
1.  需要操作者是【组织拥有者】【组织管理员】   
2. 需要操作者是  被删除项目的【项目超级管理员】

   
需要同时满足 两个条件。才可对该项目进行删除操作。
{% endhint %}



刪除项目位于 组织管理- 组织项目中 。

‌  
点击头像，进入组织概览 。
